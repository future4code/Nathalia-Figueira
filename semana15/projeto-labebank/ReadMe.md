# Sistema Bancário

Estrutura de Criação de Conta do Usuário:
|   * CPF (Único)
    * Nome
    * Data de Nascimento ( >= 18)
 
Estrutura de Armazenamento de Dados do Usuário:
    * CPF (Único)
    * Nome
    * Data de Nascimento ( >= 18)
    * Saldo (Valor Inicial = 0)
    * Extrato
        - Valor
        - Data
        - Descrição
